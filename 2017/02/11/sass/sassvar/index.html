Sass 声明多值变量

多值变量分为 List 类型和 Map 类型，简单来说 List 类型有点像 JavaScript 中的数组，而 Map 类型有点像 JavaScript 中的对象。

List

List 类型的数据可通过空格，逗号或小括号分隔多个值，可用 nth( $var, $index ) 取值。关于 List 类型的数据操作还有很多其他函数如 zip( $lists… )、 nth( $list, $n )、 index( $list, $value ) 等，具体可参考 Sass Functions（搜索 List Functions 即可）。

Map

Map 类型数据以 Key 和 Value 成对出现，其中 Value 又可以是 List 。格式为： $map: ( key1: value1, key2: value2, key3: value3 );。可通过 map-get( $map, $key ) 取值。关于 Map 类型数据还有很多其他函数如 map-remove( $map, $keys… )、 map-has-key( $map, $key )、 keywords( $args ) 等，具体可参考 Sass Functions（搜索 Map Functions 即可）。
